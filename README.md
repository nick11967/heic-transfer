# HowTo
1. Create python virtualenv with `python3 -m venv .venv` and activate it.
2. Run `pip install -r requirements.txt`.
3. Create `./HEIC` directory by `mkdir HEIC`. And put your .HEIC files in it.
4. Run `python heic2jpg.py` to transfer HEIC file to JPEG. Currently only transfer to JPEG is possible.
